## 📌 Overview

**Bharat Rights Twin (BRT)** is a public-interest governance infrastructure that transforms how citizens interact with welfare systems.

It connects:

```
Policy → Eligibility → Exclusion → Legal Action → Citizen Signals → Governance Intelligence
```

India has built strong delivery systems (DBT, Aadhaar, scheme portals), but a critical gap remains:

> Citizens don’t just need access to schemes — they need **clarity, action, and remedy**.

BRT is designed to fill that gap.

---

## 🎯 The Problem

| System            | Limitation                                    |
| ----------------- | --------------------------------------------- |
| Scheme Portals    | Inform but don’t detect exclusion             |
| Grievance Systems | Accept complaints but don’t guide action      |
| Legal Tools       | Exist but are inaccessible                    |
| Policy Reports    | Identify gaps but don’t create feedback loops |

📉 Result:
**Delivery success in data, but failure in lived experience**

---

## 🧩 The Solution

A unified platform where a citizen can:

✔ Understand eligibility
✔ Identify exclusion
✔ Generate legal action (RTI / grievance / notice)
✔ Contribute anonymised field signals

## 🔬 Policy MRI (Core Innovation)

> **“Where schemes succeed on paper but fail in practice.”**

Policy MRI converts citizen experiences into:

* 📍 district-level intelligence
* 📊 failure pattern detection
* 🧠 real-time governance insights

This creates a **citizen-verified delivery layer**, independent of administrative reporting.
